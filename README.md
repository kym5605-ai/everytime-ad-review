# 에브리타임 광고 소재 검수 도구

에브리타임 광고주가 소재를 전달하기 전에 **규격을 셀프 검수**하고 **실제 게재면 미리보기**를 확인하는 팀 전용 웹 도구입니다.

- 지면별 자동 검수(사이즈·형식·용량·투명배경·글자수·이모지·특수문자·영상 길이/해상도)
- 게재 미리보기 합성 + JPG 다운로드, 입력 텍스트 실시간 렌더
- 반려 사항 발생 시 소재제작가이드 페이지를 인용한 **메일 회신용 문구 자동 생성**

## 접근

**팀 전용 페이지입니다.** 접속 시 공용 비밀번호가 필요합니다(사내 채널로 공유). 페이지 전체가 클라이언트에서 AES-256-GCM으로 암호화되어 있어, 비밀번호 없이는 소재·내용을 볼 수 없습니다.
